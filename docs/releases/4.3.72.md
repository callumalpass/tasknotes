# TaskNotes 4.3.72 (Fork)

*This release is from the [cybersader/tasknotes](https://github.com/cybersader/tasknotes) fork.*

## Fixed

- **"Assign from column" reliably writes values** -- Formula column values are now pre-resolved when you click "Add", while Bases entries are still live. Eliminates stale reference issues over SMB/network vaults.
- **Frontmatter fallback for non-formula columns** -- Assignment reads from frontmatter directly as a safety net when Bases data is unavailable.
