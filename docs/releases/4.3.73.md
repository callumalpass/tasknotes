# TaskNotes 4.3.73 (Fork)

*This release is from the [cybersader/tasknotes](https://github.com/cybersader/tasknotes) fork.*

## Fixed

- **Upcoming View respects remapped due dates** -- Tasks with `tnDueDateProp` (e.g., `next_assessment_due` mapped to Due) now show in the correct time category instead of "No due date". The same applies to scheduled dates via `tnScheduledDateProp`.
- **PropertyPicker discovers all properties over SMB** -- The property picker (in both Convert/Generate and View Settings tabs) now uses pre-extracted frontmatter from items instead of relying solely on the metadata cache. Fixes the issue where only `dateCreated` and `dateModified` appeared on slow network vaults.
